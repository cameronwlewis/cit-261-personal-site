Alright, let's talk about manipulating this here DOM!

First, we're gonna talk about how to use createElement. Using createElement means that we must specify the kind of element we want within the parentheses, as follows:

document.createElement("div");

But line of code doesn't really accomplish anything on it's own, so we need to store that value into a variable:

var newDiv = document.createElement("div");

Great! Now we can accomplish something with it. This element we've created is empty, so let's put something in it. How about some text?

var newContent = document.createTextNode("Stop clicking me!!!!");

Perfect, we've created a 'text node' to put into our div element we created earlier. Now let's add this text node to our div element with this:

//add the text node to the newly created div.
newDiv.appendChild(newContent);

To understand how appendChild works, just think of a child and a parent. A 'child' in HTML is simply the nested element within another element, like this:

<div id="parent">
<span id="sassy_child"></span>
</div>

The only problem we have now is that we need a place to insert our newDiv with it's text node child. Here we can use insertBefore to insert it into our HTML document. First though, we'll need to tell our function where we'd like to place it. See the following code:

// add the newly created element and its content into the DOM
var currentDiv = document.getElementById("manip");
document.body.insertBefore(newDiv, currentDiv);

Great, now our text will appear when we run this function that we've packaged all this code in.

function add_text() {
// create a new div element
// and give it some content
var newDiv = document.createElement("div");
var newContent = document.createTextNode("Stop clicking me!!!!");

// add id for later retrieval, and a class name for styling purposes
newDiv.id = "new-text";
newDiv.className = "demo--result";

//add the text node to the newly created div.
newDiv.appendChild(newContent);

// add the newly created element and its content into the DOM
var currentDiv = document.getElementById("insert-before-here");
document.body.insertBefore(newDiv, currentDiv);
}

And if we wish to remove our text node? Test it out in the demo area!

function remove_text(){
var parent = document.getElementById("my-page");
parent.removeChild(document.getElementById("new-text"));
}
